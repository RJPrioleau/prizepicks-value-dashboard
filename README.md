# PrizePicks Value Dashboard

A Python decision-support tool for analyzing PrizePicks props using projections, market edges, matchup adjustments, risk ratings, confidence scoring, and historical performance tracking.

## Current Features

- Load props from CSV
- Calculate projection edge
- Calculate market edge
- Apply matchup adjustments
- Generate suggestions
- Rate risk
- Score confidence
- Classify play type
- Save suggestion history
- Prevent duplicate saved plays
- Show history summary
- Show sport, confidence, and play type breakdowns
