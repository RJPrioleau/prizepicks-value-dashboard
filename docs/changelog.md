# Changelog

All notable changes to the PrizePicks Value Dashboard project will be documented in this file.

## Version 0.4.0 - Prop Comparison Engine

### Added

* Prop comparison engine
* Multi-prop ranking system
* CSV prop loading
* More-side opportunity reporting
* Less-side opportunity reporting
* Actionable opportunity detail reporting
* Risk type architecture support

### Improved

* Recommendation confidence logic
* Prop ranking display
* Tie-breaker logic using hit rate
* Historical analysis reporting

### Documentation

* Added section headers throughout historical_stats.py
* Added beginner-friendly docstrings
* Added Lo Development Standard comments and notes

### Architecture

* Separated analysis logic from display/report logic
* Added support for NORMAL, GOBLIN, and DEMON risk types
* Introduced CSV-driven workflow for prop analysis
